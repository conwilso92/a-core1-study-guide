# CompTIA A+ Core 1 (220-1201) Study Guide

A comprehensive study guide for the CompTIA A+ Core 1 (220-1201) certification exam, synthesized from course materials and official exam objectives. Built as a single dark-mode HTML file with full anchor navigation.

🔗 **[View the Study Guide](https://conwilso92.github.io/aplus-core1-study-guide)**

## Coverage

All 16 sections mapped to the 5 official exam domains:

| Domain | Weight | Sections |
|---|---|---|
| 1. Mobile Devices | 15% | Sec. 14 |
| 2. Networking | 20% | Sec. 9, 10, 11, 12 |
| 3. Hardware | 25% | Sec. 2, 3, 4, 5, 6, 7, 8, 15 |
| 4. Virtualization & Cloud Computing | 11% | Sec. 13 |
| 5. Hardware & Network Troubleshooting | 29% | Sec. 1, 12, 14, 15 |

### What's Inside

- **Section 1 — IT Fundamentals & Troubleshooting:** CompTIA's 7-step troubleshooting methodology with step-by-step numbered flow, role of an IT specialist
- **Section 2 — Cables, Connectors & Interfaces:** USB standards and speeds (1.1 through USB4), USB connector types, Thunderbolt 1–5, video connectors (VGA, DVI-A/D/I, HDMI versions, DisplayPort), SATA/eSATA/M.2/Molex/IDE interfaces, active vs passive adapters
- **Section 3 — Motherboards:** Form factors (ATX, mATX, Mini-ITX, E-ATX) with dimensions, CPU sockets (LGA/PGA/ZIF), key components (DIMM slots, PCIe slots, M.2, chipset, CMOS battery, front panel headers), expansion cards, installation order
- **Section 4 — Power Supplies & Cooling:** PSU types (modular, semi-modular, non-modular, redundant), 80 PLUS efficiency ratings, all connector types with wattage ratings, cooling systems (heat sink, thermal paste, AIO liquid cooling, case airflow)
- **Section 5 — Processors:** x86/x64/ARM architecture, RISC vs CISC, multi-core, Hyper-Threading/SMT, virtualization support (VT-x/AMD-V), cache levels (L1/L2/L3), TDP, Intel/AMD CPU families, integrated graphics
- **Section 6 — System Memory:** DDR3/4/5 specs with voltage and pin counts, DIMM vs SO-DIMM, single/dual/quad channel, ECC vs non-ECC, RDIMM vs UDIMM, virtual memory and page file
- **Section 7 — Storage Devices:** HDD vs SATA SSD vs NVMe SSD speed comparison, RPM speeds and latency, SSD wear leveling and TBW endurance, RAID 0/1/5/6/10 with capacity math, removable storage types
- **Section 8 — BIOS/UEFI & Boot Process:** BIOS vs UEFI comparison table, key firmware settings (Secure Boot, TPM, XMP/EXPO, VT-x), POST and beep codes, MBR vs GPT partition tables
- **Section 9 — Networking Hardware:** Network types (LAN/WAN/WLAN/MAN/PAN/SAN), network devices (hub, switch, router, WAP, firewall, proxy, UTM, IDS/IPS), managed vs unmanaged switches, PoE standards (802.3af/at/bt) with wattage and use cases
- **Section 10 — Network Cabling:** Cat 5 through Cat 8 with speeds, distances, and standards, UTP/STP/F-UTP/S-FTP shielding types, T568A and T568B full pin-by-pin color tables, straight-through vs crossover, cable tools (crimper, punchdown, toner probe, loopback, TAP), fiber optic (SMF vs MMF), fiber connectors (LC/SC/ST), coaxial cable (RG-6/RG-59, F-type, BNC), copper cable problems (attenuation, crosstalk, EMI, opens, shorts)
- **Section 11 — IP Addressing & Network Config:** Internet connection types (DSL, cable, fiber, cellular, satellite), IPv4 addressing (classful, private ranges, APIPA, loopback, subnet mask), IPv6 (format, shortening rules, link-local, global unicast), TCP vs UDP, complete port reference table, DHCP, DNS record types (A/AAAA/CNAME/MX/PTR/SOA/NS/TXT), VLANs, VPN, NAT/PAT
- **Section 12 — Network Services:** Server types with ports, proxy/UTM/IDS/IPS/SCADA/IoT devices, wired and wireless connectivity troubleshooting, VoIP issues (latency, jitter, QoS)
- **Section 13 — Virtualization & Cloud:** Type 1 vs Type 2 hypervisors with examples, virtualization use cases (sandboxing, legacy support, containers), cloud deployment models (public/private/community/hybrid), cloud service models (IaaS/PaaS/SaaS) with target audiences
- **Section 14 — Mobile Devices:** Laptop components (SO-DIMM, battery, display, wireless NIC, digitizer, biometrics, NFC), mobile connectivity (Wi-Fi 802.11 standards, Bluetooth, cellular 4G/5G, GPS, NFC, airplane mode), mobile troubleshooting (swollen battery, overheating, screen issues, malware indicators), disassembly best practices
- **Section 15 — Printers & Multifunction Devices:** Printer types (laser, inkjet, thermal, impact, 3D), laser printing 7-step process (Processing → Charging → Exposing → Developing → Transferring → Fusing → Cleaning), troubleshooting table with symptoms and fixes, maintenance by type, connectivity and sharing options
- **Section 16 — Security Fundamentals:** CIA Triad, identity and access management (identification/authentication/authorization/accounting), security principles (least privilege, implicit deny, zero trust, defense in depth), MFA factor categories, Windows Hello, SSO, OTP, hard vs soft tokens, NTFS vs share permissions with effective permissions, encryption (hashing, symmetric, asymmetric, digital signatures, TPM, HSM, BitLocker)
